# Detailed-Sales-Report-using-Power-BI

## 📖 Overview

The **Executive Sales Report Dashboard** is an interactive Power BI report designed to provide a comprehensive view of sales performance, customer activity, product performance, and payment trends.

This dashboard enables stakeholders to monitor key business metrics, identify high-performing categories and products, analyze customer purchasing behavior, and track sales trends over time.

## 🚀 Features

### 📈 Sales by Category
- Displays total sales across product categories.
- Identifies top-performing categories.
- Supports category-level performance analysis.

### 📦 Product Sales Analysis
- Shows sales generated by individual products.
- Highlights best-selling and underperforming products.

### 💳 Payment Type Distribution
- Visualizes sales distribution by payment method.
- Compares revenue generated through:
  - Check
  - Credit Card
  - Cash
  - Credit Transfer

### 👥 Customer Performance Analysis
Provides customer-level insights including:
- Total Sales
- Maximum Sales
- Minimum Sales
- Average Sales
- Transaction Count

### 📅 Monthly Sales Trend
- Tracks sales performance across months.
- Helps identify seasonal trends and revenue fluctuations.

---

## 📊 Key Metrics

| Metric | Description |
|----------|-------------|
| Total Sales | Total revenue generated |
| Average Sales | Average sales per customer |
| Maximum Sales | Highest sales value |
| Minimum Sales | Lowest sales value |
| Transaction Count | Number of sales transactions |
